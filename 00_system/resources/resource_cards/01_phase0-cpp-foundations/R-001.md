# R-001 Card for Phase 0

## Resource
- ID: `R-001`
- Title: LearnCpp.com - "Learn C++" tutorials
- Role in repo: Primary reading path for `Phase 0: C++ Foundations`
- Best use: First explanation whenever a Phase 0 concept is covered well enough for learner-facing use

## Gate 0 Packet: Compile, Run, and Basic I/O
Read in this order:

| Step | Read this | Direct link | Why this step matters |
|------|-----------|-------------|------------------------|
| 1 | Introduction to the compiler, linker, and libraries | https://www.learncpp.com/cpp-tutorial/introduction-to-the-compiler-linker-and-libraries/ | Gives the basic mental model of what happens before a program runs |
| 2 | Compiling your first program | https://www.learncpp.com/cpp-tutorial/compiling-your-first-program/ | Connects source file, compile action, and running the built program |
| 3 | Configuring your compiler: Warning and error levels | https://www.learncpp.com/cpp-tutorial/configuring-your-compiler-warning-and-error-levels/ | Explains why warnings are part of the work and not optional cleanup |
| 4 | Introduction to iostream: cout, cin, and endl | https://www.learncpp.com/cpp-tutorial/introduction-to-iostream-cout-cin-and-endl/ | Covers the exact I/O model needed for `Gate 0: Compile, Run, and Basic I/O` |

## Gate 1 Packet: Stack, Heap, and Storage Basics
Read in this order:

| Step | Read this | Direct link | Why this step matters |
|------|-----------|-------------|------------------------|
| 1 | Local variables | https://www.learncpp.com/cpp-tutorial/local-variables/ | Gives the basic model for block-local variables and automatic storage duration |
| 2 | Dynamic memory allocation with new and delete | https://www.learncpp.com/cpp-tutorial/dynamic-memory-allocation-with-new-and-delete/ | Explains what `new` creates, what `delete` does, and why leaks happen |
| 3 | The stack and the heap | https://www.learncpp.com/cpp-tutorial/the-stack-and-the-heap/ | Connects the duration model to the common stack/heap vocabulary used in this gate |
| 4 | Scope, duration, and linkage summary | https://www.learncpp.com/cpp-tutorial/scope-duration-and-linkage-summary/ | Optional follow-up only when the names `automatic`, `static`, and `dynamic` are still unclear after steps 1-3 |

## Gate 2 Packet: Values, References, and Parameter Passing
Read in this order:

| Step | Read this | Direct link | Why this step matters |
|------|-----------|-------------|------------------------|
| 1 | Introduction to function parameters and arguments | https://www.learncpp.com/cpp-tutorial/introduction-to-function-parameters-and-arguments/ | Gives the base model for what the caller passes and what the function receives |

## Gate 3 Packet: First Class, Lifetime, and Basic RAII

|------|-----------|-------------|------------------------|
