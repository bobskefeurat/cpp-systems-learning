## Gate 0 Packet: Compile, Run, and Basic I/O
Use this packet only when the learner does not understand parts of the baseline command:

```powershell
g++ main.cpp -std=c++20 -Wall -Wextra -Wpedantic -o main
```

Read only the needed part:

| Need to understand | Read this | Direct link | Stop when you can explain |
|--------------------|-----------|-------------|----------------------------|
| what `g++` is doing | Invoking G++ | https://gcc.gnu.org/onlinedocs/gcc/Invoking-G_002b_002b.html | why `g++` is used to build a C++ program |
| what `-o main` means | Overall Options | https://gcc.gnu.org/onlinedocs/gcc/Overall-Options.html | what output file naming does in this command |
| what `-std=c++20` means | C++ Dialect Options | https://gcc.gnu.org/onlinedocs/gcc/C_002b_002b-Dialect-Options.html | that the flag selects the language standard used for compilation |
| what `-Wall -Wextra -Wpedantic` mean | Warning Options | https://gcc.gnu.org/onlinedocs/gcc/Warning-Options.html | why the command enables extra diagnostics and what those flags are for |

## Scope Control for Gate 0
This manual is not the main teaching path for `Gate 0: Compile, Run, and Basic I/O`.

Use it:
- to clarify a specific flag
- to confirm what a GCC option means

Do not use it as the learner's first explanation if `R-001` is enough.
